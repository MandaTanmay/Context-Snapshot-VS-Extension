# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-06-06

### Added
- Initial release of Context Snapshot extension
- Command to capture context snapshot (`contextSnapshot.capture`)
- Keybinding support (Ctrl+Shift+Alt+S on Windows/Linux, Cmd+Shift+Alt+S on macOS)
- Capture all open files with full content and cursor positions
- Collect diagnostics (errors and warnings) across workspace
- Capture workspace metadata and VS Code version
- Generate markdown-formatted snapshot
- Copy snapshot to clipboard
- Error handling and user feedback messages
